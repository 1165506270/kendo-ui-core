Basic Usage of DropDownList

Source element

<input id="input" />

Initialization

$("#input").kendoDropDownList([
    {text: "Item1", value:"1"},
    {text: "Item2", value:"2"},
    {text: "Item3", value:"3"}
]);

Source element

<select id="select">
    <option>Item1</option>
    <option>Item2</option>
    <option>Item3</option>
</select>

Initialization

$("#select").kendoDropDownList();
