    Quickly Learn Kendo

    The Kendo demos are designed to help you quickly learn how to use the complete Kendo framework. The following demo selections are available:

    Framework
    Learn how to use Kendo DataSource, Templates and Drag & Drop
    UI Widgets
    Learn how to use each of the Kendo UI widgets, including quick API reference
    Integration
    Learn how to use Kendo with other popular Javascript libraries such as jQuery Templates and KnockoutJS

    Aeroviewr

    This rich HTML5 integration demo shows Kendo in action to interact with Flickr photo streams. The demo features several Kendo UI controls, Kendo DataSource, and other Kendo framework elements. Check it out online and in mobile browsers to see how Kendo helps create universally accessible apps.

    Mobile Device Compatibility

    Kendo UI is built to work seamlessly across desktop browsers and mobile touchscreen devices. Kendo UI offers full support for touch interactions and is tested against the following mobile platform browsers:

    • Android 2.x+
    • iOS 3.x+
    • BlackBerry OS 6.x+
    • webOS X.x+

    Supported browsers

    • Internet Explorer 6+
    • Firefox 3+
    • Safari 4+
    • Chrome
    • Opera 10+
