Kendo UI

Kendo UI is a jQuery-based framework for building modern web apps. The framework features more than 10 UI components, including a Grid and Chart, and all of the tools needed for a HTML5 app development, such as Data Binding, Templating, Drag-and-Drop API, and more.

Installing and Getting Started with Kendo UI

Kendo UI is a pure JavaScript library, so installing and getting started is quick and easy:

  1. Download Kendo UI
  2. Copy the Kendo UI CSS and JavaScript resources to your project
  3. Configure your page to use Kendo UI: 

    <!--In the header of your page, paste the following for Kendo styles-->
<link href="kendo.common.css" rel="stylesheet" type="text/css" />
<link href="kendo.kendo.css" rel="stylesheet" type="text/css" />

<!--Before the closing BODY tag, paste the following script references-->
<script src="jquery-1.6.2.js" type="text/javascript"></script>
<script src="kendo.all.js" type="text/javascript"></script>

  4. That's it! You're ready to experience the power and simplicity of Kendo UI. Explore the demos to see how quickly you can begin building rich HTML5 applications with Kendo.

Quickly Learn Kendo

The Kendo demos are designed to help you quickly learn how to use the complete Kendo framework. The following demo sections are available:

  • Framework: Learn how to use Kendo DataSource, Templates, and Drag & Drop
  • UI Components: Learn how to use each of the Kendo UI components, including quick API reference
  • Integration: Learn how to use Kendo with other popular JavaScript libraries, like jQuery Templates and KnockoutJS

Kendo UI is built to work seamlessly across desktop browsers and mobile touchscreen devices. Kendo UI offers full support for touch interactions and is tested against the following mobile platform browsers:

  • Android 2.x+
  • iOS 3.x+
  • BlackBerry OS 6.x+
  • webOS X.x+
