• ajaxRequest (pane, url, data)
    Loads the pane content from the specified URL.

    Example

    // get a reference to the splitter
var splitter = $("#splitter").data("kendoSplitter");
// load pane content
splitter.ajaxRequest("#Panel1", "/customer/profile", { id: 42 });

    Parameters

    pane: Selector | DOM Element
    The pane whose content should be loaded.
    url: String
    The URL which returns the pane content.
    data: Object | String
    Data to be sent to the server.
  • collapse (pane)
    Collapses the specified Pane item

    Example

    // get a reference to the splitter
var splitter = $("#splitter").data("kendoSplitter");
// collapse the specified pane
splitter.collapse("#Item1"); // id of the first pane

    Parameters

    pane: Selector | DOM Element
    The pane, which will be collapsed.
  • expand (pane)
    Expands the specified Pane item

    Example

    // get a reference to the splitter
var splitter = $("#splitter").data("kendoSplitter");
// expand the pane
splitter.expand("#Item1"); // id of the first pane

    Parameters

    pane: Selector | DOM Element
    The pane, which will be expanded.
  • max (pane, value)
    Set the maximum size of the pane.

    Example

    // get a reference to the splitter
var splitter = $("#splitter").data("kendoSplitter");
// set the max size of the pane
splitter.max("#Item1", "300px");

    Parameters

    pane: Selector | DOM Element
    The pane
    value: String
    The maximum size value.
  • min (pane, value)
    Set the minimum size of the pane.

    Example

    // get a reference to the splitter
var splitter = $("#splitter").data("kendoSplitter");
// 
splitter.min("#Item1", "100px");

    Parameters

    pane: Selector | DOM Element
    The pane
    value: String
    The minimum size value.
  • size (pane, value)
    Set the size of the pane.

    Example

    // get a reference to the splitter
var splitter = $("#splitter").data("kendoSplitter");
// set the size of the pane
splitter.size("#Item1", "200px");

    Parameters

    pane: Selector | DOM Element
    The pane
    value: String
