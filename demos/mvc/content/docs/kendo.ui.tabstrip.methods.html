• append (tab)
    Appends a TabStrip item to the end of the tab list.

    Example

    tabStrip.append(
    [{
        text: "Item 1",
        url: "http://www.kendoui.com"               // Link URL if navigation is needed, optional.
    },
    {
        text: "Item 2",
        content: "text"                             // Content for the content element
    },
    {
        text: "Item 3",
        contentUrl: "partialContent.html"           // From where to load the item content
    },
    {
        text: "Item 4",
        imageUrl: "http://www.kendoui.com/test.jpg" // Item image URL, optional.
    },
    {
        text: "Item 5",
        spriteCssClass: "imageClass3"               // Item image sprite CSS class, optional.
    }]
);

    Parameters

    tab: Selector
    Target tab, specified as a JSON object. You can pass tab text, content or contentUrl here. Can handle an HTML string or array of such strings or JSON.
  • contentElement (itemIndex)
    Returns the contentElement of a an item by its index.

    Parameters

    itemIndex: int
    Target tab index
  • enable (element, enable)
    Enables/disables TabStrip tab/s

    Parameters

    element: Selector
    Target element
    enable: Boolean
    Desired state

  • remove (element)
    Removes the specified TabStrip item/s

    Example

    Parameters

    element: Selector
    Target item selector.
  • select (element)
    Selects the specified TabStrip tab/s. If called without arguments - returns the selected tab.

    Example

    Parameters

    element: Selector
    Target tab selector.
