        <div style="display: block;" id="codeStrip-1" class="description t-content t-state-active"><p>
     The Menu widget displays hierarchical data as a multi-level menu. Menus provide
     rich styling for unordered lists of items, and can be used for both navigation and
     executing JavaScript commands. Items can be defined and initialized from HTML, or
     the rich Menu API can be used to add and remove items.
 </p>
 <h3>Getting Started</h3><div class="code-sample"><h4 class="code-title">Create a simple HTML hierarchical list of items</h4><pre class="code prettyprint"><code><span class="tag">&lt;ul</span><span class="pln"> </span><span class="atn">id</span><span class="pun">=</span><span class="atv">"menu"</span><span class="tag">&gt;</span><span class="pln"><br>&nbsp; &nbsp; </span><span class="tag">&lt;li&gt;</span><span class="pln">Item 1<br>&nbsp; &nbsp; &nbsp; &nbsp; </span><span class="tag">&lt;ul&gt;</span><span class="pln"><br>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span><span class="tag">&lt;li&gt;</span><span class="pln">Item 1.1</span><span class="tag">&lt;/li&gt;</span><span class="pln"><br>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span><span class="tag">&lt;li&gt;</span><span class="pln">Item 1.2</span><span class="tag">&lt;/li&gt;</span><span class="pln"><br>&nbsp; &nbsp; &nbsp; &nbsp; </span><span class="tag">&lt;/ul&gt;</span><span class="pln"><br>&nbsp; &nbsp; </span><span class="tag">&lt;/li&gt;</span><span class="pln"><br>&nbsp; &nbsp; </span><span class="tag">&lt;li&gt;</span><span class="pln">Item 2</span><span class="tag">&lt;/li&gt;</span><span class="pln"><br></span><span class="tag">&lt;/ul&gt;</span></code></pre></div><div class="code-sample"><h4 class="code-title">Initialize Kendo Menu using jQuery selector</h4><pre class="code prettyprint"><code><span class="kwd">var</span><span class="pln"> menu </span><span class="pun">=</span><span class="pln"> $</span><span class="pun">(</span><span class="str">"#menu"</span><span class="pun">).</span><span class="pln">kendoMenu</span><span class="pun">();</span></code></pre></div><h3>Customizing Menu Animations</h3>
 <p>
     By default, the Menu uses a slide animation to expand and reveal sub-items as the
     mouse hovers. Animations can be easily customized using configuration properties, changing
     the animation style and delay. Menu items can also be configured to open on click instead of on hover.
 </p><div class="code-sample"><h4 class="code-title">Changing Menu animation and open behavior</h4><pre class="code prettyprint"><code><span class="pln">$</span><span class="pun">(</span><span class="str">"#menu"</span><span class="pun">).</span><span class="pln">kendoMenu</span><span class="pun">({</span><span class="pln"><br>&nbsp; &nbsp; &nbsp;animation</span><span class="pun">:</span><span class="pln"> </span><span class="pun">{</span><span class="pln"><br>&nbsp; &nbsp; &nbsp; &nbsp;open </span><span class="pun">:</span><span class="pln"> </span><span class="pun">{</span><span class="pln">effects</span><span class="pun">:</span><span class="pln"> fadeIn</span><span class="pun">},</span><span class="pln"><br>&nbsp; &nbsp; &nbsp; &nbsp;hoverDelay</span><span class="pun">:</span><span class="pln"> </span><span class="lit">500</span><span class="pln"><br>&nbsp; &nbsp; &nbsp;</span><span class="pun">},</span><span class="pln"><br>&nbsp; &nbsp; &nbsp;openOnClick</span><span class="pun">:</span><span class="pln"> </span><span class="kwd">true</span><span class="pln"><br>&nbsp;</span><span class="pun">});</span></code></pre></div><h3>Dynamically configuring Menu items</h3>
  <p>
         The Menu API provides several methods for dynamically adding or removing Items.
         To add items, provide the new item as a JSON object along with a reference item that
         will be used to determine the placement in the hierarchy.
 </p>
 <br>
 <p>
         A reference item is simply a target Menu Item HTML element that already exists in
         the Menu. Any valid jQuery selector can be used to obtain a reference to the target
         item. For examples, see the <a title="Menu API demos" href="../menu/api.html">Menu API demos</a>.
         Removing an item only requires a reference to the target element that should be removed.
