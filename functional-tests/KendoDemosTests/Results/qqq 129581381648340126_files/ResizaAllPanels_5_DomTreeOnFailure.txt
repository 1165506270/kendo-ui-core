        <div style="display: block;" id="codeStrip-1" class="description t-content t-state-active"><p>
      The Splitter widget provides an easy way to create a dynamic layout of resizable and
      collapsible panes. The widget converts the children of an HTML element in to the interactive
      layout, adding resize and collapse handles based on configuration. Splitters can be mixed
      in both vertical and horizontal orientations to build complex layouts.
  </p>
  <h3>Getting Started</h3><div class="code-sample"><h4 class="code-title">Create a root HTML div element with children that will become panes</h4><pre class="code prettyprint"><code><span class="tag">&lt;div</span><span class="pln"> </span><span class="atn">id</span><span class="pun">=</span><span class="atv">"splitter"</span><span class="tag">&gt;</span><span class="pln"><br>&nbsp; &nbsp;</span><span class="tag">&lt;div&gt;</span><span class="pln"><br>&nbsp; &nbsp; &nbsp; &nbsp;Area 1<br>&nbsp; &nbsp;</span><span class="tag">&lt;/div&gt;</span><span class="pln"><br>&nbsp; &nbsp;</span><span class="tag">&lt;div&gt;</span><span class="pln"><br>&nbsp; &nbsp; &nbsp; &nbsp;Area 2<br>&nbsp; &nbsp;</span><span class="tag">&lt;/div&gt;</span><span class="pln"><br></span><span class="tag">&lt;/div&gt;</span></code></pre></div><div class="code-sample"><h4 class="code-title">Initialize the Splitter using a jQuery selector</h4><pre class="code prettyprint"><code><span class="pln">&nbsp; $</span><span class="pun">(</span><span class="str">"#splitter"</span><span class="pun">).</span><span class="pln">kendoSplitter</span><span class="pun">();</span></code></pre></div><p>
      When the Splitter is initialized, a vertical split bar will be placed between the two
      HTML divs. This bar can be moved by a user left and right to adjust the size on the panes.
  </p>
  <h3>Configuring Splitter Behavior</h3>
  <p>
      Splitter provides many configuration options that can be easily set during initialization.
      Among the properties that can be controlled:
  </p>
  <ul>
      <li>Min/Max pane size</li>
      <li>Resizable and Collapsible pane behaviors</li>
      <li>Orientation of the splitter</li>
  </ul>
  <p>
      Pane properties are set for each individual pane in a Splitter, whereas Splitter properties apply to the entire widget.
